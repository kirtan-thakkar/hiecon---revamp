# Google Antigravity Design System

You are NOT building a dashboard.

You are building a premium editorial website.

## Design Inspiration

Primary reference:

- Google Antigravity
- Google AI
- Google Gemini
- Apple
- Stripe
- Vercel
- Framer

Never generate generic shadcn card layouts.

---

## Layout Philosophy

Prioritize:

Whitespace

Typography

Photography

Motion

Editorial composition

NOT cards.

Every section must feel different.

Never repeat

3 cards

3 cards

3 cards

throughout the page.

---

## Cards

Avoid cards unless absolutely necessary.

Prefer:

split layouts

editorial layouts

full-width imagery

feature strips

horizontal storytelling

sticky scroll

immersive sections

---

## Motion

Animations should feel invisible.

Use Motion.

Use GSAP.

Use:

opacity

blur

translateY

clip-path reveal

text reveal

image reveal

parallax

sticky sections

horizontal scroll

animated counters

Never use:

bounce

spring everywhere

overscaled hover

crazy rotations

---

## Typography

Use typography as the primary visual hierarchy.

Large headlines.

Small labels.

Comfortable spacing.

Editorial rhythm.

---

## Spacing

Use generous spacing.

Sections should breathe.

Avoid stacking components closely together.

---

## Components

When using shadcn

Prefer:

Separator

Accordion

Carousel

Hover Card

Command

Popover

instead of wrapping everything inside Card.

---

## Color

Use theme variables.

Always use:

bg-background

bg-card

text-foreground

text-muted-foreground

border-border

Never write:

dark:bg-black

dark:text-white

dark:border-zinc-800

The application already manages themes.

---

## Images

Large.

Cinematic.

Edge-to-edge where appropriate.

Never tiny thumbnails.

---

## Hero

Should feel like a product launch.

Large headline.

Minimal text.

Large industrial imagery.

Subtle motion.

---

## Product Section

Avoid grids of identical cards.

Experiment with:

horizontal scrolling

large showcase

alternating layouts

magazine layouts

bento only when meaningful

---

## Overall Goal

Every page should feel handcrafted.

The result should resemble Google's Antigravity website rather than a dashboard generated by AI.
